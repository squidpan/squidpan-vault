## Install openvpn on linux (Pop!_OS 22.04 LTS)

Following steps from [Install OpenVPN 3 Linux](https://community.openvpn.net/Pages/OpenVPN3Linux)


1. Ensure you have the needed support packages already installed:

```code
apt install apt-transport-https curl
```


## Retrieve the OpenVPN Inc package signing key:

```code
mkdir -p /etc/apt/keyrings    ### This might not exist in all distributions
sudo su ### need to be root to avoid Permission denied error" when redirecting output to /etc file
curl -sSfL https://packages.openvpn.net/packages-repo.gpg >/etc/apt/keyrings/openvpn.asc

exit ### exit root
apt update
```

## Replace the `DISTRIBUTION` part in the command below using the release name from the table above to set up the apt source listing:

for Ubuntu 22.04 LTS - release name is __jammy__
```code
sudo su ### need to be root to avoid Permission denied error" when redirecting output to /etc file

echo "deb [signed-by=/etc/apt/keyrings/openvpn.asc] https://packages.openvpn.net/openvpn3/debian jammy main" >>/etc/apt/sources.list.d/openvpn3.list

exit ### out of root

sudo apt update
apt install openvpn3-client
```
